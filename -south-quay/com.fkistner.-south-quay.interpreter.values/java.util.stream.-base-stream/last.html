last

val <T, S : BaseStream<T, S>> BaseStream<T, S>.last: T?

Efficiently gets the last value of a stream by splitting its underlying spliterator until only one element (or close to) is left.

The spliterator is required to be Spliterator.ORDERED, Spliterator.SIZED, and Spliterator.SUBSIZED.

T - Element type

S - Stream type

Return
